# Image Processing Studio Pro

A GUI-based Image Processing application built using Python and OpenCV.

## Features

- Gaussian Filter
- Median Filter
- Mean Filter
- FFT Filter
- Wavelet Denoising
- Bilateral Filter
- Guided Filter
- CNN Image Processing
- Morphological Operations
- Noise Simulation

## Technologies Used

- Python
- OpenCV
- NumPy
- Tkinter
- PyTorch
- Scikit-Image
- PyWavelets

## Installation

```bash
pip install -r requirements.txt
```

Run the project:

```bash
python main.py
```
